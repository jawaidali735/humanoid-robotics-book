# API Documentation

This section contains the API specifications for the educational content management system of the Physical AI and Humanoid Robotics book.

## Available APIs

- [Content API](./content-api.yaml) - API for managing educational content including chapters, exercises, and code examples

## API Overview

The educational content API provides endpoints for managing the various components of the book content:

- **Chapters**: Endpoints for creating, retrieving, updating, and deleting book chapters
- **Modules**: Endpoints for organizing content into the four core modules
- **Exercises**: Endpoints for managing hands-on exercises
- **Code Examples**: Endpoints for handling executable code examples
