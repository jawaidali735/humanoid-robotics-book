# Module 1: The Robotic Nervous System (ROS 2)

Welcome to Module 1 of the Physical AI & Humanoid Robotics guide. This module covers the fundamentals of Robot Operating System 2 (ROS 2), which serves as the nervous system for humanoid robots.

## Overview

This module will teach you:
- Core ROS 2 concepts and architecture
- Middleware, nodes, topics, services, and actions
- Practical implementation with rclpy Python integration
- URDF humanoid descriptions
